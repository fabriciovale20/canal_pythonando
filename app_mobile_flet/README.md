Código para cadastro de produtos

Ferramentas utilizadas:
- Linaguem: Python
- Biblioteca: Flet

# Criar Banco de Dados
Para realizar a criação do banco de dados é necessário rodar o código 'models.py'

Em seguia será gerado o arquivo 'appflet.db'

# Funções da Aplicação
- Cadastro de produto (Descrição e Preço)
- Listagem de todos os produtos cadastrados
- Alerta de sucesso ou falha de cadastro de produto

# Execução do código
1. pip install -r requirements.txt
2. Execute o arquivo ``app.py``
